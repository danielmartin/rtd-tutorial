# Getting Started with Core

Follow this tutorial to set up Core.

This is some C++ code:

```c++
#include <iostream>

int main() {
   std::cout << "Hello, world!\n";
}
```

Some TypeScript:

```typescript
let message: string = 'Hello, World!';
console.log(message);
```

Some Ruby:

```ruby
class HelloWorld
    def initialize(name)
       @name = name.capitalize
    end
    def sayHi
       puts "Hello #{@name}!"
    end
 end

 hello = HelloWorld.new("World")
 hello.sayHi
```
